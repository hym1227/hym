# 时代少年团（Teens in Times、TNT） 是由时代峰峻推出的中国内地男子演唱组合，由马嘉祺、丁程鑫、宋亚轩、刘耀文、张真源、严浩翔、贺峻霖七人组成 [1]  。
##2019年11月23日，举行出道暨新歌首唱会，时代少年团组合正式出道 [2]  ；12月6日，获2020爱奇艺尖叫之夜年度潜力组合称号；12月8日，获TMEA腾讯音乐娱乐盛典年度新声代团体称号 [3]  ；12月31日，出席2020湖南卫视跨年演唱会 [4]  。2020年7月19日，出席第27届东方风云榜 [5]  ，并获最佳新锐组合称号 [32]  ；12月10日，为电影《晴雅集》演唱的同名主题曲上线 [16]  ；12月20日，获腾讯视频星光大赏年度大势组合称号 [33]  ；12月28日，发行的单曲《2035是这young》上线 [31]  ；12月31日，出席东方卫视跨年盛典 [34]  。2021年2月5日，参加录制的综艺《接招吧！前辈》播出 [39]  ；2月11日，登上2021年中央广播电视总台春节联欢晚会的舞台，表演歌曲《奔跑的青春》 [26-27]  ；2月28日，获2020新浪微博之夜微博年度人气团体称号 [29]  。代少年团
